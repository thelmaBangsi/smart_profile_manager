# smart_profile_manager
A Python program to manage user profiles with age categories and multi-user support.

Description

A Python command-line application that collects user information, categorizes users as Adult or Minor based on age, stores multiple profiles, and displays a formatted summary.

Features

Collects user name, age, height, country, and favorite color

Categorizes age automatically (Adult / Minor)

Supports multiple user profiles in one session

Displays profiles in a clean, multi-line summary
